# ✨ NextGen AI Chatbot: S-Tier AI Workspace

A beautifully designed, high-performance AI chat interface built with Next.js. NextGen AI transforms standard LLM interactions into a powerful, agentic workspace featuring live code execution, real-time web browsing, continuous voice conversations, and professional productivity tools.

## 🌟 Premium Features

### 🧘 Zen Mode & Distraction-Free Workspace
- **Deep Focus**: Instantly hide all sidebars and UI distractions to focus entirely on the conversation.
- **Activation**: Use the global keyboard shortcut or the sleek UI toggle.
- **Responsive Layout**: Fluidly adapts to any screen size, providing a localized center-staged experience.

### ⌨️ Power-User Command Palette (`Ctrl + K`)
- **Global Hub**: Quickly switch models, jump to past conversations, or trigger specialized actions without taking your hands off the keyboard.
- **Smart Search**: Real-time filtering system for instant access to your entire chat history.

### 📑 Intelligent Thread Summarization
- **Context at a Glance**: Long conversation? Trigger a "Summarize" action to get a concise 3-5 point takeaway generated by AI.
- **Efficiency**: Perfect for returning to deep research threads after a break.

### 📦 Pro Snippet Library
- **Reusable Components**: Store frequently used prompts, code snippets, or templates in a dedicated library.
- **One-Click Insert**: Instant insertion into the chat bar, saving time and mental load.

### 🎙️ Multilingual Voice Engine (Whisper v3)
- **Auto-Detection**: Speak naturally in **English, Telugu, Hindi**, or any supported language. The system automatically identifies and transcribes the spoken language.
- **Centered UI**: A beautiful, minimal red-wave animation provides real-time feedback during recording.
- **Auto-Send**: Detects silence to automatically process and send your voice messages.

### 🌐 Agentic Web Search
- **Live Knowledge**: Toggle web search to force the AI to research real-time information across the internet.
- **Persistence**: Your search preference is saved across sessions for a customized permanent workflow.

### 💻 Developer Tools & Live Sandbox
- **Live Code Sandbox**: Execute **JavaScript** or **Python** (via Pyodide) directly in the integrated Artifact Viewer.
- **Markdown Excellence**: Full GFM support with smart code blocks that auto-collapse for better readability.

## ⌨️ Global Keyboard Shortcuts

| Shortcut | Action |
| :--- | :--- |
| `Ctrl + K` | Open Command Palette |
| `Ctrl + .` | Toggle Zen Mode |

## 🛠️ Tech Stack

*   **Framework**: [Next.js](https://nextjs.org/) (App Router, Turbopack)
*   **Styling**: [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) & [Tailwind CSS](https://tailwindcss.com/)
*   **Animations**: [Framer Motion](https://www.framer.com/motion/)
*   **Icons**: [Lucide React](https://lucide.dev/)
*   **Database & Auth**: [Supabase](https://supabase.com/) (PostgreSQL + RLS)
*   **AI Engine**: Google Gemini 2.5, Groq (Llama 3 / Mixtral), Ollama (Local AI)
*   **Voice**: Groq Whisper-large-v3
*   **Execution**: Pyodide (Python) & Sandboxed Iframe (JS)

## ⚙️ Local Development Setup

### 1. Prerequisites
- Node.js 18.x or later
- A Supabase project
- API Keys: Google Gemini, Groq

### 2. Environment Configuration
Create a `.env.local` file:
```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
GEMINI_API_KEY=your_gemini_api_key
GROQ_API_KEY=your_groq_api_key
```

### 3. Quick Start
```bash
npm install
npm run dev
```
